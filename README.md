# Anime Recommendation For Beginners

This is a simple anime recommendation system for beginners.

## Features

- Anime starter pack
- Recommend anime based on genre
  - Action
  - Adventure
  - Comedy
  - Drama
  - Fantasy
  - Romance
  - Sci-Fi
  - Slice of Life
  - Suspense

## Usage

1. Clone this repository
2. Run `yarn install`
3. Run `yarn start`
